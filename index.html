Handige commands in linux

Commands

    cd: Change Directory

    Gebruik je om naar een andere directory te gaan.

    ls: List Directory

    Toont de bestanden en mappen in de huidige directory.

    mkdir: Make Directory

    Maakt een nieuwe map aan.

    rm: Remove

    Verwijdert bestanden of mappen.

    cp: Copy

    Kopieert bestanden of mappen van de ene locatie naar de andere.

    mv: Move

    Verplaatst of hernoemt bestanden of mappen.

    touch: Create File

    Maakt een nieuw leeg bestand aan.

    pwd: Print Working Directory

    Toont het pad van de huidige directory.

    cat: Concatenate and Display

    Toont de inhoud van een bestand.

    man: Manual

    Toont de handleiding voor een ander command.

    less: Less.

    Toont de inhoud van een bestand.

    head: Head

    Toont de eerste regels van een bestand.

    tail: Tail

    Toont de laatste regels van een bestand.

    nano: Nano

    Opend een teksteditor in de terminal.

    exit: Exit

    Sluit de terminal af.

    sudo: Sudo

    Voert een command uit met administratorrechten.

    snap: Snap

    Beheert snap-pakketten op je systeem.

    apt-get: APT Get

    Beheert softwarepakketten op je systemen.

    tree: Tree

    Toont de directorystructuur in een boomvorm.
